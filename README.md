# Social Media Campaign Performance Tracker

This project analyzes the performance of social media ad campaigns
using basic digital marketing metrics.

## Dataset
The dataset includes impressions, clicks, and ad spend
for multiple campaigns.

## Metrics Calculated
- Click Through Rate (CTR)
- Cost Per Click (CPC)

## Tools Used
- Microsoft Excel
- GitHub

## Outcome
The analysis helps identify high-performing campaigns
based on engagement and cost efficiency.
